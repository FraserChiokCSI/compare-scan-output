# Wafw00f

## Tool Purpose
[Wafw00f](https://github.com/EnableSecurity/wafw00f) detect the existence of firewalls.

### Relavent Usage
```
subfinder -d  -o <output.txt>
wafw00f -a <top-level domain (TLD)> -o <output.json>
```
